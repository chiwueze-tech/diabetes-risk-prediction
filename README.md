# diabetes-risk-prediction
Predicting diabetes risk using supervised machine learning, including data cleaning, EDA, and model development on healthcare data.

🩺 Diabetes Risk Prediction (Supervised Machine Learning)
📌 Overview

This project is a beginner-friendly machine learning project that predicts whether a patient is likely to have diabetes based on medical data.

It covers the full data science workflow:

Data cleaning
Exploratory Data Analysis (EDA)
Feature analysis
Model building

The project is designed to demonstrate how machine learning can be applied in healthcare for early detection of diseases.

📊 Dataset

The dataset contains medical attributes such as:

Glucose level
Blood pressure
BMI (Body Mass Index)
Age
Insulin level
Skin thickness
Diabetes pedigree function

Target Variable:

Outcome
0 → Non-diabetic
1 → Diabetic
🧹 Data Cleaning
Replaced invalid values (e.g., 0s in medical fields) with NaN
Filled missing values using the mean
Ensured dataset consistency for analysis
📈 Exploratory Data Analysis (EDA)

Key insights from the analysis:

Glucose levels are more spread out and higher in diabetic patients
Diabetic patients tend to have higher average glucose levels
There is a strong relationship between glucose level and diabetes outcome
